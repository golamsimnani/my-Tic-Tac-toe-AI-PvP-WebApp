# my-Tic-Tac-toe-AI-PvP-WebApp
An advanced implementation of Tic Tac Toe engineered to break the solved state of the traditional 3 \times 3 matrix. This project introduces scalable grid sizes (3 \times 3, 4 \times 4, 5 \times 5) fully integrated into an omni-channel gameplay machine featuring multi-tier AI execution, local PvP state management, and an endless survival loop.
